# n8n Gold Price Alert Automation

This project is a simple automation built with **n8n** that retrieves the **gold price from the Binance API** and automatically sends the result via **email**.

The workflow demonstrates how to connect external APIs and automate notifications using n8n.

---

## 🚀 Features

- Fetch real-time gold price using **Binance API**
- Process API response in n8n
- Send automatic **email notifications**
- Simple and reusable workflow

---

## 🛠 Workflow Overview

The automation flow works as follows:

1. Trigger the workflow
2. Send HTTP request to Binance API
3. Retrieve gold price data
4. Format the response
5. Send email notification
